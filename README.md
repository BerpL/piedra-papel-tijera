## Descripción
Juego interactivo de Piedra, Papel o Tijera con animaciones realistas que simulan el movimiento físico del juego tradicional. Las manos se sacuden tres veces antes de revelar la elección, recreando la experiencia del juego en la vida real.

## Reglas del Juego
1. **Piedra** vence a **Tijera**
2. **Tijera** vence a **Papel**
3. **Papel** vence a **Piedra**

## Cómo Jugar
1. Abre el archivo `index.html` en tu navegador web
2. Haz clic en uno de los tres botones: **Piedra**, **Papel** o **Tijera**
3. Las manos se sacudirán 3 veces simulando el ademán del juego físico
4. Se revelará tu elección y la de la computadora con una animación
5. Se mostrará el resultado de la ronda (Ganaste, Perdiste o Empate)
6. El marcador se actualiza automáticamente
7. Presiona el botón **Reiniciar** para comenzar un nuevo juego

## Características
- Animaciones fluidas que simulan el juego físico
- Efecto de sacudida de manos (3 veces)
- Animación de revelación al mostrar las elecciones
- Emojis visuales para cada opción
- Sistema de puntuación persistente durante la sesión
- Interfaz moderna y responsive
- Efectos visuales para victoria, derrota y empate
- Botones deshabilitados durante las animaciones

## Tecnologías Utilizadas
- **HTML5**: Estructura del juego
- **CSS3**: Estilos y animaciones
- **JavaScript**: Lógica del juego y manejo de eventos

## Instalación y Uso
1. Clona o descarga este repositorio
2. No se requieren dependencias adicionales
3. Abre el archivo `index.html` en cualquier navegador moderno
4. ¡Comienza a jugar!

## Características Visuales
- **Gradiente de fondo**: Diseño moderno con colores púrpura y violeta
- **Animaciones CSS**: Transiciones suaves y efectos de rebote
- **Responsive Design**: Adaptable a diferentes tamaños de pantalla
- **Feedback visual**: Colores distintivos para cada resultado (verde=victoria, rojo=derrota, amarillo=empate)

## Notas
- La animación completa dura aproximadamente 2.3 segundos
- Los botones se deshabilitan automáticamente durante las animaciones para evitar clicks múltiples
- El juego funciona completamente del lado del cliente (no requiere servidor)
